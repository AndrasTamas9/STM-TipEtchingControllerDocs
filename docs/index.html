STM Tip Etching Controller – Technical Documentation

Overview

The STM Tip Etching Controller is a fully custom-designed and semi-automated
electrochemical system for producing high-quality platinum–iridium STM tips,
using a 1 molar CaCl₂ aqueous electrolyte.

Purpose of the Device

This system automates the electrochemical etching of Pt/Ir STM tips, providing:

  • Highly reproducible apex geometry
  • Real-time RMS current measurement with filtering
  • Automatic detection of meniscus contact using current thresholds
  • Multi-stage etching logic (contact → validation → etching → retract)
  • Dual-relay controlled voltage switching (9 V and 30 V stages)
  • Precision microstepping motion using a TMC2209 driver
  • Manual jogging mode for fine z-axis control
  • On-device parameter editor via LCD keypad
  • Automatic baseline current calibration at Z = 30 mm

The device increases reproducibility, reduces operator intervention,
and provides reliable tip quality for STM experiments.

Software Architecture

Firmware components:

  • CurrentSensor – ADC acquisition, RMS and peak processing
  • StepperDriver – non-blocking motion controller
  • ModeController – main UI state machine
  • Modes
    • HOME – homing & baseline calibration
    • MOD1 – standard etching
    • MOD2 – pulsed etching
    • JOG – manual stepping
    • PARAM – configuration editor
  • KeypadShield – analog button interface
  • Lcd1602 – display & backlight control
  • MovingAverage – optimized fixed-point filter
  • Parameters – persistent runtime configuration

All modules run non-blocking, ensuring real-time acquisition and stable motor control.
