Detailed Description

Implementation of a fixed-point, template-based moving average filter.

This file provides the method definitions for the MovingAverage template, which implements an efficient sliding window average using a fixed-size circular buffer and integer (fixed-point) accumulation for improved numeric stability on microcontrollers.

The template parameters are:

  • N : window length (number of samples),
  • SCALE : fixed-point scaling factor applied to the input values.
